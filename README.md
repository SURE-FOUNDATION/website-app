# SFGS Website (React + Vite)

## Setup
- Install: `npm install`
- Dev server: `npm run dev`
- Lint: `npm run lint`
- Build: `npm run build`
- Preview build: `npm run preview`

## Assets
- Public assets live in `public/images` and `public/docs`.
- Centralized asset paths are defined in `src/constants/assets.js`.
